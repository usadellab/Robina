Release Notes

1.2.4_build647


Welcome to RobiNA and thank you for using it to evaluate and analyse
your microarray and RNA-Seq data. Before taking off please take the time to read the
release notes carefully.

  • New workflow for RNA-Seq based transcript profiling: Check out the new workflow
    for RNA-Seq based analysis of differential gene expression. To-date it supports
    import ofIllumina/Solexa-type raw sequence data in FASTQ format, SAM/BAM prealigned
    reads and precomputed counts tables.

  • Treatment of replicates: The current Version of RobiNA assumes that
    all replicates that you enter are true biological replicates. Technical
    replication is not yet taken into account. When entering technical replicates
    the significance of differential expression will be overestimated leading
    to an artificially increased number of genes that are significantly called
    differentially expressed.
  • Unconnected designs in two color experiments: Comparing results from two
    sets of two color microarrays that are not connected requires analysing the
    color channels separately. This is not yet supported in RobiNA.
