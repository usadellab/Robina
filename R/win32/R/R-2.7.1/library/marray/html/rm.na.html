rm.na {marray}R Documentation

Remove missing values

Description

Remove NA's, NAN's and INF's from a vector.

Usage

rm.na(x)

Arguments

x A numeric vector

Value

A vector with all NA's remove.

Author(s)

Jean Yang

Examples

x <- round(rnorm(10), 2)
x[c(2,4,5)] <- NA
x
rm.na(x)
