m.spot {limma}R Documentation

Extract M or A-values from SPOT data.frame or matrix

Description

Extract M-values or A-values from a SPOT data.frame or matrix.

These functions are deprecated in limma 2.9.7, 1 January 2007.

Usage

m.spot(spot)
a.spot(spot)

Arguments

spot data.frame or matrix giving SPOT output for one microarray

Value

Vector of M-values (m.spot) or A-values (a.spot)
