AffyBatch instance affybatch.example

Description

This is an artifical data set. It contains two similar examples (affybatch.example, affybatch.example2) with values subset from the Dilution dataset to make two 100x100 chips. Either example is suitable for exploring affy; both are used to demonstrate merge.AffyBatch.

Format

An AffyBatch of three samples.
